# Contributing to homebrew-tools

Thank you for your interest in contributing to this project!

## Reporting Bugs

Bugs and issues should be reported via [GitHub Issues](https://github.com/tenstorrent/homebrew-tools/issues).

Please include:
- A clear description of the problem
- Steps to reproduce the issue
- Your macOS or Linux version and Homebrew version (`brew --version`)
- Any relevant error output

## Submitting Changes

Bug fixes and new functionality should be submitted via [Pull Requests](https://github.com/tenstorrent/homebrew-tools/pulls).

- Pull requests are reviewed on a **weekly** basis.
- Keep changes focused and minimal — one logical change per PR.
- Update `README.md` if your change affects user-facing behavior.
- If bumping the Tracy formula version, use `scripts/bump_tracy_formula.py` rather than editing `Formula/tracy.rb` by hand.

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this standard.
